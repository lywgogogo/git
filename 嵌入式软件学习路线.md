## 大一暑假（2026.7–2026.8）项目

### 智能环境监测器（FreeRTOS + 云平台）

**材料**（< 50 元）：

```
DHT22 温湿度传感器、BH1750 光照传感器
0.96寸 OLED 屏、ESP8266 WiFi 模块
面包板、杜邦线
```

**架构**：

```
┌─────────────┐    I2C     ┌──────────┐
│ BH1750      │──────────→│          │
└─────────────┘           │          │
┌─────────────┐  单总线   │ STM32    │──→ OLED 显示
│ DHT22       │──────────→│ FreeRTOS │
└─────────────┘           │          │
                          │  任务1: 传感器采集 (10ms)
┌─────────────┐   UART    │  任务2: 数据处理   (50ms)
│ ESP8266     │←─────────│  任务3: OLED显示   (100ms)
└──────┬──────┘           │  任务4: MQTT上传   (1s)
       ↓ MQTT             └──────────┘
   ┌────────┐
   │ 云平台  │ → 手机查看
   └────────┘
```

**用到的技术**：FreeRTOS 多任务、队列、I2C、单总线、UART、MQTT、Git

做完推 GitHub，README 放效果视频 + 接线图。**这个项目是后面找实习的核心弹药。**

## 大三上（2027.9–2028.1）方向收敛

此时你应该知道自己更偏好哪个方向：

### A：Linux 驱动/BSP（薪资最高）

- 深入内核：进程调度、内存管理、VFS
- 深入驱动：platform、设备树、中断子系统
- 目标：BSP/驱动工程师

### B：RTOS + IoT（机会最多）

- 多 RTOS：FreeRTOS、ThreadX、Zephyr
- 无线协议栈：BLE Mesh、Zigbee、LoRa、WiFi
- 低功耗设计
- 目标：嵌入式软件工程师

### C：边缘 AI（最前沿，但要慎重）

- TensorFlow Lite Micro、CMSIS-NN
- 目标：AIoT 工程师
- **注意**：岗位少，学历门槛可能更高

**建议**：A 为主，B 保底。Linux 驱动的薪资天花板比 MCU 高 30-50%。
